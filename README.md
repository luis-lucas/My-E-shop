
The Noodle Shop — E-commerce Frontend
A responsive e-commerce landing page built with HTML, CSS and JavaScript while learning web development from scratch.
Features:

Responsive navbar with animated hamburger menu
Typing animation hero section
Product cards with add to cart functionality and cart counter
Testimonial slider
Newsletter form with email validation
Sticky header and smooth scroll
Announcement banner with close button

Built with: HTML, CSS, JavaScript
Learning approach: Built step by step as a learning project, using AI as an interactive tutor for guidance and hints. All code was written by me — the AI explained concepts and pointed me in the right direction without writing the code directly.
